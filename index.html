Projects

Caravan Boat Design

The Caravan Boat Design project involved the conceptualization, design, and implementation of a novel caravan boat. The aim was to create a versatile vessel that could serve both as a recreational vehicle on land and a fully functional boat on water.

Key Features and Achievements:

  • Innovative design combining the functionality of a caravan and a boat.
  • Utilized advanced materials for lightweight and durable construction.
  • Integrated state-of-the-art navigation and safety systems.
  • Conducted extensive testing and optimization to ensure stability and performance.
  • Collaborated with a multidisciplinary team of engineers and designers.

Technologies and Tools Used:

  • CAD software (AutoCAD, SolidWorks) for detailed design and modeling.
  • CFD simulations to analyze fluid dynamics and optimize hull design.
  • Finite Element Analysis (FEA) for structural integrity assessment.
  • 3D printing for prototype development and testing.
  • Project management tools to streamline collaboration and timelines.

This project demonstrated my ability to integrate various engineering disciplines, innovate within design constraints, and deliver a product that meets both aesthetic and functional requirements.

Unmanned Surface Vehicle

The Unmanned Surface Vehicle (USV) project focused on designing and developing an autonomous vessel capable of performing various maritime tasks without human intervention. This project aimed to enhance maritime safety and operational efficiency through automation.

Key Features and Achievements:

  • Developed an autonomous navigation system for precise control and maneuverability.
  • Incorporated advanced sensors for obstacle detection and environmental monitoring.
  • Designed a robust communication system for real-time data transmission and remote control.
  • Implemented machine learning algorithms for adaptive route planning and decision making.
  • Conducted rigorous testing in simulated and real-world conditions to validate performance.

Technologies and Tools Used:

  • ROS (Robot Operating System) for developing the autonomous navigation system.
  • Various sensors including LiDAR, sonar, and GPS for environmental sensing.
  • Python and C++ for implementing control algorithms and data processing.
  • Machine learning frameworks such as TensorFlow for adaptive decision making.
  • Simulation tools like Gazebo for testing and validation.

This project showcased my expertise in autonomous systems, sensor integration, and machine learning, contributing to advancements in unmanned maritime technology.
